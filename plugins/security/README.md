# security plugin

Security skills for code review, cloud infrastructure, and Claude Code configuration auditing.

## Skills

| Skill | Description |
|-------|-------------|
| `security-review` | Comprehensive security checklist for auth, user input, secrets, API endpoints, and payment flows |
| `security-scan` | Scan Claude Code configuration (`.claude/`) for misconfigurations, injection risks, and hook vulnerabilities |

## Install

```sh
claude plugin add marketplace github:mmalyska/claude-plugins
claude plugin install security@mmalyska/claude-plugins
```
